Very Useful Spells Indeed

Very Useful Spells Indeed is a micro zine that was created as an entry to Micro Zine Jam 2020. It is a singe A4 page that can be printed out an folded into a small 8 page zine.
It has spurred me on to start creating the Very Useful Library Indeed which will be a collection of micro zines filled with spells and other wonderful things.
