nnn file browser

This is a good light weight terminal file browser.

Setting up

Enable cd on quit.

{ Quit on cd files }

Create the file ./config/fish/functions/n.fish and paste the contents of the cd on quit file for fish file. Remove the -x from the export statments so that ^G is the only wat to cd on quit. Start nnn as just n for cd on quit to work.

Set defaults.

Set some default options in your .bashrc . 
EDITOR=micro   # edit files using micro when e is pressed
PAGER=less     # preview files using less
